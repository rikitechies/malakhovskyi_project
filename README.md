# БудОблік: Система обліку витрат на будівництво

Веб-додаток для обліку будівельних матеріалів та контролю фінансових витрат.
## Основний функціонал
* **Довідник матеріалів:** Додавання, перегляд та видалення будівельних матеріалів із розбиттям на категорії.
* **Облік витрат:** Фіксація транзакцій (дата, матеріал, кількість, загальна вартість).
* **Управління даними (CRUD):** Можливість не лише додавати та видаляти, а й **редагувати** існуючі записи про витрати.
* **Пошук та фільтрація:** Швидкий текстовий пошук за назвою або категорією, а також **фільтрація витрат за обраний період (дати)**.
* **Експорт в Excel:** Генерація та завантаження фінансового звіту у форматі `.xlsx`.
